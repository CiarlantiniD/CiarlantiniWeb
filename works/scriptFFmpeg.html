Script FFmpeg

Descripción

Script basado en Bash para realizar transcodificaciones de diferentes tipos de archivos con FFmpeg en terminal de Linux.

Características

- Soporte para formato AVI, MOV, MXF.
- Transcoficicación de video a diferentes formatos.
- Extracción de Thumbnail en PNG.
- Concatación de videos.
- Sistema de LOG.
- Cálculo estimado de tiempo para los procesos.
